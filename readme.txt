Create a highly customizable slider with images, videos, text, overlay, buttons and a fully customizable layout

== Description ==

This plugin is great for creating multiple sliders with ease. Create as many slides as you want, set them to a group and display on any page or post 
using the shortcode [scslider slider="demo-slider"]
